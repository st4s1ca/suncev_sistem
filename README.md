# suncev_sistem
Projekat koji simbolizira Sunčev sistem i opisuje svaku od planeta i neke nebeske fenomene.
Ovaj repozitorijum sadrži drugi projekat koji sam radila u toku školovanja i koji predstavlja osnovne principe html, css i js i odražava početničko znanje i upoznavanje sa izdraom veb stranica 
Projekat je implementiran u toku školovanja u elektrotehničkoj školi "Mihajlo Pupin" u okviru završnog projektnog zadatka, za predmet veb programiranje u četvrtoj godini srednjoškolskog obrazovanja.
**Vreme implementacija projekta je maj 2024. godine ali je projekat upload-ovan na github repozitorijum kasnijeg datuma.
